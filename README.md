# 🏦 Banco R7 (Java)

Este é um projeto simples desenvolvido em Java com o objetivo de praticar lógica de programação e conceitos básicos da linguagem.

## 📌 Funcionalidades

- Cadastro de usuário
- Login (simulado)
- Menu interativo no terminal
- Transferência de saldo
- Consulta de saldo

## 🧠 Objetivo

O projeto foi criado com fins educacionais. A ideia é evoluí-lo continuamente conforme novos conceitos forem sendo aprendidos, como:

- Programação Orientada a Objetos (POO)
- Validações mais robustas
- Persistência de dados
- Interface gráfica (futuramente)

## 🚀 Tecnologias utilizadas

- Java
- IntelliJ IDEA

## 📈 Evolução

Este projeto será atualizado constantemente como parte da minha jornada de aprendizado em desenvolvimento Java.
